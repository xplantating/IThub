8 страниц с работающей логикой

Страницы: 
1) Главная страница - страница с новостями
2)Страница обучения
3) Курс IT-Волонтера
4) Доска объявлений
5) Личный кабинет
6) Страница Обмен очков на мерч (магазин)
7) Страница входа 
8) Страница регистрации
